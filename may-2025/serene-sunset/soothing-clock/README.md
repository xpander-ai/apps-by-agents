# Soothing Clock

A minimal web clock that changes the background color smoothly based on the current time, offering a soothing visual experience.

## Features
- Live digital clock display (HH:MM:SS)
- Background color transitions smoothly throughout the day
- Built with pure HTML, CSS, and JavaScript (no frameworks)
- Responsive and visually calming

## How it works
- The background color is generated using the current time, mapped to the HSL color wheel for a full spectrum, with lightness modulated for a soft effect.
- The color changes every second, creating a gentle, ever-shifting ambiance.

## Usage
Just open `index.html` in your browser. No build step required.
