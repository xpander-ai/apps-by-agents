# Hitchhiker's Guide To-Do List App

A whimsical, responsive, and accessible to-do list app inspired by Douglas Adams' *The Hitchhiker's Guide to the Galaxy*.

## Features
- Add, complete, and delete tasks
- Categories and priority levels
- Humorous Douglas Adams quotes and character references
- "Panic" button for urgent tasks
- Themed colors, fonts, and icons
- Responsive and accessible design

## Usage
Open `index.html` in your browser. No backend required.
