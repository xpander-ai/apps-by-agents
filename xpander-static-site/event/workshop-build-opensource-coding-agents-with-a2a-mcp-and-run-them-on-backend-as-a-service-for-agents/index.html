AI Agent Workshop: build opensource coding agents with A2A & MCP and run them on Backend-as-a-Service for agents

This workshop is for developers and AI agent builders who are interested in building and running advanced agents.

​With native MCP client support in your IDE, you can handle everything – tasks, CI/CD, dashboards, logs, configs – without jumping between 5 different tools. No more tab overload, just do everything from your IDE by calling AI Agents.

​We’ll use the first Backend for agents platform – xpander.ai – to build and host a powerful setup with multiple AI Agents, no-effort tool integrations, State and memory, MCP triggering, A2A agent handoffs, and more. ​In under three hours, you’ll leave with working AI Agents you can apply to your real development workflows.
